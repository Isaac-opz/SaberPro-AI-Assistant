Pregunta:¿Cómo puedo consultar mi usuario contraseña temporal ya que no me llegó al correo?
Respuesta:Validarlo con el programa académico de lo contrario contactarse a las líneas de atención, ingresando a la  página www.icfes.gov.co / Atención y servicios a la ciudadanía / canales de atención, línea nacional Bogotá: (601) 484-1460 o a través del chat. 

Pregunta:¿Cómo puedo consultar mi usuario contraseña temporal?
Respuesta:Verificar en la base de  datos de los Usuarios y contraseñas cargada en la plataforma del conocimiento. Si al iniciar una convocatoria no se cuenta con la base de usuarios y contraseñas, es necesario validar el caso con el gestor, para confirmar como proceder. 

Pregunta:Si cuenta con el usuario y contraseña y no permiten ingresar ¿Qué debo hacer?
Respuesta:Consultar en la base de usuario y contraseñas, si los datos están correctos y al realizar las auditorías nos permite ingresar, se orienta al ciudadano como ingresar los datos para realizar la personalización del usuario, que tenga en cuenta las mayúsculas y minúsculas, números e ingresar la información completa. En caso que no permita el ingreso con el usuario y contraseña que está en la base, y verificando que no haya personalizado el usuario, se le debe informar que radique solicitud con el pantallazo del error, adjuntando el documento e indicando el correo electrónico que desea asociar. Para comunicaciones escritas, después de realizar las auditorías correspondientes, se escala el caso a la DTI.

Pregunta:Si al personalizar el usuario me aparece un mensaje que el correo ya está registrado ¿Qué debo hacer?
Respuesta:Validar en prisma si la ciudadana tiene ese correo asociado a otro documento, en caso de ser así, se le puede indicar que para garantizar el registro ingreso un correo diferente, en caso que informe que no tiene otro correo y que quiere que sea el inicial, se le debe decir que radique petición, adjuntando el documento e informando el correo ya que es unificación de usuarios. Para comunicaciones escritas, después de realizar las auditorías correspondientes, se escala el caso a la DTI.

Pregunta:¿Donde es posible consultar el estado de un estudiante, Autorizado Preinscrito e Inscrito?
Respuesta:Para confirmar si ya está autorizado o Preinscrito la primera consulta se realizar a través de auditoría de Registro de PRISMA, en caso que se requiera consultar si se encuentra inscritos se debe ingresar por medio de módulo de gestión de PRISMA. En caso de no encontrarlo por PRISMA, se realiza la consulta en la base de datos de los estados publicada en la plataforma del conocimiento.

Pregunta:¿Qué respuesta le entregamos a un ciudadano que solicita corregir los datos registrados durante el proceso de registro al Examen Saber T y T?
Respuesta:(Estudiantes e individuales): Es importante señalar que, si presenta alguna inconsistencia en el nombre o en el número de documento de identidad, podrá realizar dichas modificaciones durante el proceso de registro a través de la plataforma PRISMA. No obstante, la información sólo podrá ser modificada una vez siempre y cuando esta no supere el 30% de los caracteres. Para realizar los cambios antes mencionados, debe ingresar a la opción modificaciones, seleccionar Corregir nombre, documento o cambio de municipio e ingresar los datos actualizados.

Pregunta:¿Cómo realizar la corrección de datos fuera de las fechas de registro?
Respuesta:Radicar una solicitud por escrito, adjuntado los documentos pertinentes para la corrección e indicando el dato errado y el dato correcto. Para comunicaciones escritas, después de realizar las auditorías correspondientes, se escala el caso a la DTI.

(Revisar base: Listado Aspirantes SENA Responsables y NO Responsables de PAGO) En caso que Usted ingrese y el sistema le genere cobro debe radicar solicitud por escrito con el pantallazo del cobro generado. Los alumnos que hayan sido reportados como beneficiarios SENA y que formen parte del convenio para el año 2021-1/2, ya deben estar asociados como estudiantes exentos de pago por parte de dicha Institución. (Revisar base: Listado Aspirantes SENA Responsables y NO Responsables de PAGO) En caso que no se encuentre en las base o no contemos con la base de beneficiaros se debe remitir al ciudadano con el SENA, teniendo en cuenta que dicha Institución es quien debe autorizarlo..
