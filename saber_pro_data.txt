11
Pregunta*******************************************************************************************
Si cuenta con el usuario y contraseña y no permiten ingresar ¿Qué debo hacer?
****************************************************************************************************
Respuesta******************************************************************************************
Consultar en la base de usuario y contraseñas, si los datos están correctos y al realizar las auditorías nos permite ingresar, se orienta al ciudadano como ingresar los datos para realizar la personalización del usuario, que tenga en cuenta las mayúsculas y minúsculas, números e ingresar la información completa. En caso que no permita el ingreso con el usuario y contraseña que está en la base, y verificando que no haya personalizado el usuario, se le debe informar que radique solicitud con el pantallazo del error, adjuntando el documento e indicando el correo electrónico que desea asociar. Para comunicaciones escritas, después de realizar las auditorías correspondientes, se escala el caso a la DTI. 
****************************************************************************************************

Pregunta*******************************************************************************************

Para comunicaciones escritas, después de realizar las auditorías correspondientes, se escala el caso a la DTI. 
****************************************************************************************************

14
Pregunta*******************************************************************************************
¿Qué respuesta le entregamos a un ciudadano que solicita corregir los datos registrados durante el proceso de registro al Examen Saber T y T?
****************************************************************************************************
Respuesta******************************************************************************************
(Estudiantes e individuales): Es importante señalar que, si presenta alguna inconsistencia en el nombre o en el número de documento de identidad, podrá realizar dichas modificaciones durante el proceso de registro a través de la plataforma PRISMA. No obstante, la información sólo podrá ser modificada una vez siempre y cuando esta no supere el 30% de los caracteres. Para realizar los cambios antes mencionados, debe ingresar a la opción modificaciones, seleccionar Corregir nombre, documento o cambio de municipio e ingresar los datos actualizados. 
****************************************************************************************************

16
Pregunta*******************************************************************************************
¿Qué respuesta le entregamos a un ciudadano que manifiesta ser beneficiario (exento de pago) SENA, sin embargo, al ingresar a PRISMA el sistema le genera cobro?
****************************************************************************************************
Respuesta******************************************************************************************
Los alumnos reportados como beneficiarios del SENA y que formen parte del convenio para el año 2021-1/2 deben estar registrados como estudiantes exentos de pago por parte de dicha institución. Para verificar esta condición, se debe revisar la base de datos "Listado Aspirantes SENA Responsables y NO Responsables de PAGO". Si al ingresar al sistema se genera un cobro, el estudiante deberá radicar una solicitud por escrito adjuntando el pantallazo del cobro generado. En caso de que el estudiante no se encuentre en la base de beneficiarios o si la base de datos no está disponible, se le debe remitir directamente al SENA, ya que esta institución es la encargada de autorizar su exención de pago. 
****************************************************************************************************

25
Pregunta*******************************************************************************************
Cómo accedo a los resultados de las pruebas SaberPro?
****************************************************************************************************
Respuesta******************************************************************************************
A partir del año 2022, la prueba Saber PRO tiene un nuevo reporte de resultados 
con el objetivo de que cada persona evaluada pueda conocer cuál fue su 
desempeño en las competencias genéricas y específicas que evalúa el examen. El 
reporte incluye la información del puntaje global y el percentil general en el que se 
encuentra el evaluado, así como también el percentil y el nivel de desempeño 
alcanzado en cada una de las competencias evaluadas y la descripción de las 
habilidades asociadas a cada nivel. (Link:https://www.icfes.gov.co/evaluaciones-icfes/resultados/)
